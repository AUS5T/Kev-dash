Report a Security Issue

Use GitHub private vulnerability reporting for security issues affecting PatchSignal.org, the PatchSignal frontend, Worker/R2 routes, or this repository.

Do not post sensitive exploit details, working payloads, private credentials, or non-public vulnerability details in public issues, pull requests, discussions, screenshots, commits, or logs.

Open a private vulnerability report on GitHub

Not Security Reports

General CVE corrections, data-source updates, and actor attribution suggestions are not security vulnerability reports for PatchSignal itself.

Those items should be handled through normal data review instead of the private vulnerability reporting channel.
