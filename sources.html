Source Overview

PatchSignal combines vulnerability data from multiple public sources to provide a more useful triage view.

Data Sources

  1. CISA Known Exploited Vulnerabilities catalog

    Used as the primary source for vulnerabilities known to be exploited. Provides CVE IDs, affected vendors/products, dates added, due dates, required actions, and ransomware-use indicators when available.

  2. National Vulnerability Database via NIST

    Used for CVSS severity, scoring, vectors, and vulnerability descriptions when available.

  3. Exploit Prediction Scoring System via FIRST

    Used for EPSS scores and percentiles to estimate exploitation likelihood.
